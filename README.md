# Trabajo-Practico4
Contenido del Proyecto

El proyecto desarrolla un sistema básico de gestión de empleados.

Archivos:

Empleado.java: Clase que modela la entidad Empleado. Incluye constructores sobrecargados, un atributo estático para el conteo de empleados y la implementación de toString(). Los métodos actualizarSalario() están sobrecargados: uno recibe un porcentaje como parámetro y el otro aplica un incremento fijo del 15%.

Main.java: Clase de prueba destinada a la creación de objetos Empleado, la verificación de los métodos de actualización salarial y la impresión del total de empleados registrados.

Ejecución

Para compilar y ejecutar el proyecto, ejecutar la clase Main.java.
